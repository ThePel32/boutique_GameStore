Frontend

    Accueil :
        Afficher une image de fond.
        Présenter les jeux récents et les promotions sous forme de cartes.

    Navigation :
        Barres de navigation permettant d'accéder à l'accueil, aux jeux, à l'espace utilisateur, au panier et à la section admin.

    Jeux :
        Liste de tous les jeux disponibles avec détails (image, nom, genre, prix).
        Option pour ajouter des jeux au panier.

    Espace Utilisateur :
        Inscription et connexion des utilisateurs.
        Affichage des informations personnelles et de l'historique des commandes.

    Panier :
        Affichage des articles ajoutés au panier.
        Gestion du panier (ajout, suppression, modification des quantités).
        Passage à la caisse.

    Admin :
        Formulaires pour ajouter, modifier et supprimer des jeux.
        Liste des jeux avec options de gestion.

Backend

    API Gestion des Jeux :
        CRUD (Create, Read, Update, Delete) pour les jeux.

    API Utilisateurs :
        Inscription, connexion et gestion des utilisateurs.

    API Panier :
        Gestion des articles dans le panier.

    API Commandes :
        Création et suivi des commandes.

Base de Données

    MySQL :
        Tables pour les utilisateurs, les jeux, les paniers et les commandes.

NoSQL

    MongoDB :
        Stockage des ventes pour des requêtes rapides.

Développement Itératif

    Accueil :
        Développement de la page d'accueil avec les jeux et les promotions.

    Jeux :
        Liste des jeux avec la fonctionnalité d'ajout au panier.

    Panier :
        Gestion du panier et passage à la caisse.

    Espace Utilisateur :
        Inscription, connexion et profil utilisateur.

    Admin :
        Interface d'administration pour gérer les jeux.

    Déploiement :
        Déploiement sur fly.io.

Responsivité

    Utiliser Bootstrap :
        Utilisation de classes Bootstrap pour assurer la responsivité sur tous les écrans.

    Media Queries :
        Ajout de media queries pour personnaliser les styles selon la taille de l'écran.

        Créer un projet HTML, CSS et JS de base.
